# Site

> Distributions page written with [hugo](https://gohugo.io/).

## Development

Install dependencies using [`yarn`](https://yarnpkg.com/), to ensure no sudden happen changes. If you don't have it, run `npm i -g yarn`.


```bash
# Install dependencies
$ yarn
$ npm start
$ open localhost:1313
```

## Build

```bash
$ npm run build
```
